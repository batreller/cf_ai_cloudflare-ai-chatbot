# Frontend - quick start

Cloudflare AI chatbot

---

### Installation

1. Clone the repository.
2. Navigate to the same folder where this README.md file is located:
   `cd cloudflare-ai-chatbot/frontend`

## Configuration

### API Endpoint

* Update the **API_URL** to point to your backend server in file:
  `public/src/config/config.js`

---

## Running the Project

### Local Development

Start the application in development mode using:

`npx wrangler pages dev public`

### Deployment

Deploy the application using:

`npx wrangler pages deploy public`
